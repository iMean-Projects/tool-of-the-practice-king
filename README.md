# LoL Dummy Setup Tool

> A personal tool to configure Practice Tool Dummies in League of Legends by matching their stats to a chosen champion + build as closely as possible.

## Overview

Select a champion, compose a build (items + rune shards), and get a recommended item/rune set to apply to the Practice Tool Dummy — so its stats reflect the real champion as accurately as possible.
